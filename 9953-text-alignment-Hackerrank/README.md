# [Text Alignment](https://www.hackerrank.com/challenges/text-alignment/problem?isFullScreen=true)
## Easy
<div class="challenge-body-html"><div class="challenge_problem_statement"><div class="msB challenge_problem_statement_body"><div class="hackdown-content"><svg style="display: none;"><defs id="MathJax_SVG_glyphs"></defs></svg><p>In Python, a string of text can be aligned <em>left, right</em> and <em>center</em>.</p>

<p><strong>.ljust(width)</strong></p>

<p>This method returns a left aligned string of length <em>width</em>.</p>

<pre><code>&gt;&gt;&gt; width = 20
&gt;&gt;&gt; print 'HackerRank'.ljust(width,'-')
HackerRank----------  
</code></pre>

<hr>

<p><strong>.center(width)</strong></p>

<p>This method returns a centered string of length <em>width</em>.</p>

<pre><code>&gt;&gt;&gt; width = 20
&gt;&gt;&gt; print 'HackerRank'.center(width,'-')
-----HackerRank-----
</code></pre>

<hr>

<p><strong>.rjust(width)</strong></p>

<p>This method returns a right aligned string of length <em>width</em>.</p>

<pre><code>&gt;&gt;&gt; width = 20
&gt;&gt;&gt; print 'HackerRank'.rjust(width,'-')
----------HackerRank
</code></pre>
